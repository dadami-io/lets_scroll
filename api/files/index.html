반가워요, 개발자님! 여기에서는 Hitomi 관련 API들을 비롯하여 우리가 제공하는 모든 API들에 대한 문서를 볼 수 있습니다.
Hitomi Gallery URL API
Hitmoi 갤러리의 ID를 이용하여 이미지 URL들을 불러옵니다. 바로 다음의 Image API와 함께 쓰실 수 있습니다.
https://scroll.reacted.xyz/list/{Gallery_ID}
요청 예시
Image API
이미지 URL의 이미지를 불러올 수 있습니다. 위의 Hitmoi Gallery URL API와 함께 쓰실 수 있습니다.
https://scroll.reacted.xyz/image?url={Image_URL}
요청 예시
Pixiv API
이미지 ID를 이용하여 Pixiv의 이미지를 가져옵니다.
https://scroll.reacted.xyz/pixiv/{Image_ID}
요청 예시
Weather API
지역명을 이용하여 Daum 날씨의 이미지를 가져옵니다. 지역명은 한글로 입력해야 합니다.
https://scroll.reacted.xyz/weather?area={Area_Name}
요청 예시
School Code API
학교명으로 학교 코드를 반환합니다. School Timetable API와 함께 쓰실 수 있습니다.
https://scroll.reacted.xyz/comsi/search?school={School_Name}
요청 예시를 제공할 수 없음
School Timetable API
학교 코드와 학년, 반의 정보로 해당 학급의 시간표를 반환합니다. School Code API, School Self-Check API와 함께 쓰실 수 있습니다.
https://scroll.reacted.xyz/comsi/timetable?id={School_Code}&g={Grade}&c={Class}
요청 예시를 제공할 수 없음
School Self-Check API
이름(실명), 생일, 자가진단 비밀 번호, 학교명, 지역명의 정보로 자가진단을 할 수 있습니다. School Code API, School Timetable API와 함께 쓰실 수 있습니다.
https://scroll.reacted.xyz/selfCheck?name=${Name}&birth=${BirthDay}&pass=${Password}&school=${School_Name}&area=${Area_Name}
