api 사용 예제 적어주세요

/list/{품번}
  품번의 이미지 링크들을 불러온다.

/image?url={이미지 url}
  (히토미에서 사용) 이미지 url의 이미지를 불러옴

/comci/search?school={학교명}
  학교명으로 학교코드들을 검색함

/comci/timetable?id={학교코드}&g={학년}&c={반}
  학교코드의 특정 학년, 반의 시간표를 가져옴
  
/weather?area={지역명}
  지역명으로 다음 날씨 이미지 가져옴
  
/pixiv/{품번}
  품번으로 픽시브 이미지 가져옴
  
/selfCheck?name=${이름}&birth=${생일}&pass=${비번}&school=${학교명}&area=${지역명}
  여러 정보들로 자가진단을 해줌
